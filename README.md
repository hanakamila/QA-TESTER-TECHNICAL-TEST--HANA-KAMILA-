# QA-TESTER-TECHNICAL-TEST--HANA-KAMILA-
QUALITY ASSURANCE TECHNICAL TEST for AYO Indonesia

Steps on how to run the script:

1. download the dataset "booking.csv" and "schedule.csv"
2. open the file QUALITY_ASSURANCE_TECHNICAL_TEST.ipynb
3. click on the "Open in Colab" button
4. click on "files"
5. upload the dataset "booking.csv" and "schedule.csv" inside the "sample_data" folder
6. click on "Table of contents"
7. click "run cells in section" on desired test case or click "run all" to run all test cases
